# NASA Space Helper

Project Idea & How it Helps Nasa:

The idea of this project is to provide NASA employees and astronauts or any user to learn more about space. It is a platform for the public to be engaged in a set of features that include:
1 - A fun list of cool facts about space and astronauts
2 - A live countdown for the next NASA mission.
3 - Tips for astronauts.


Technologies or Tools used:
- HTML, CSS, JS
- GitHub
- GitHub project planning (backlog)
- VS Code

How to Run and Use the Project:
1 - git clone https://github.com/your-username/nasa-space-helper.git
2- To open NASASpaceHelper in Visual Studio Code
3 - To open index.html
4 - Add code to all pages (HomePage, Tab Bar, Profile..)
